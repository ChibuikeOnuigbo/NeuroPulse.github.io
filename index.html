Overview

NeuroPulse Mail brings intelligent, privacy-first assistance to your inbox. It analyzes messages in real time, highlights items that require attention, and offers concise spoken summaries — all while keeping account credentials and tokens stored locally and encrypted.

Key features

  • Smart summarization: AI-powered summaries and context-aware highlights so you can read less and act faster.
  • Priority detection: Identifies important, urgent, or potential spam messages across multiple accounts.
  • Realistic voice summaries: Natural-sounding spoken summaries powered by Edge-TTS with configurable voices.
  • Multi-account support: Monitor Gmail and Outlook accounts securely, side-by-side.
  • Local, encrypted storage: Credentials and OAuth tokens stored locally (SQLite / token.json) with encryption.
  • Developer-friendly: Built with Python + Eel, using Gmail API and Microsoft Graph API for reliable access.

How it works

NeuroPulse Mail uses OAuth flows from Google and Microsoft to obtain consented access to inbox data. Tokens are retained locally and encrypted; optional AI analysis runs through configured providers (OpenRouter or Google Gemini). Voice synthesis is handled locally via Edge-TTS integration. API keys and app secrets are loaded from a local .env file for safety and separation of concerns.

Privacy-focused Designed for education, productivity, and privacy — not tracking or sale of personal data.
